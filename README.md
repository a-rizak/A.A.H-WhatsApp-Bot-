## ✨ Features
- 📩 Auto-replies to simple messages (`hi`, `hello`)  
- ⚡ Prefix-based commands (default: `!ping`)  
- 🌐 Web Dashboard:
  - `/` → status + QR page  
  - `/status` → JSON bot status  
  - `/qr.png` → WhatsApp QR pairing  
- 🔄 Auto reconnect  
- 🔒 Restrict access to specific numbers  
- ☁️ Deployable to **Heroku**, **Railway**, or **Render**
